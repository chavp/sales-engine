Bulk Import

This tool allows you to bulk create new leads.

You can upload a CSV file from your computer:

Or copy & paste your data from Excel (don't forget the header row):

You will be able to match the columns in the next step.

Do not re-upload copies of existing leads - it will create duplicates.

Tips

  • Make sure the first row of your data is a header row, i.e. contains the column names.
  • Non-US phone numbers must start with a "+" in order to be recognized properly.
  • Multiple rows with the same company name will be merged into one lead allowing you to add multiple contacts and addresses to one lead. Alternatively, you can have multiple contact or address columns in one row.

Don't hesitate to contact us if you have any issues.
